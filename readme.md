### Chaos Experiments

Ferramentas atualmente suportadas:
---
- Chaos Mesh
- Python

Tipos de experimentos atualmente suportados:
---
- Container Failure
- CPU Load test
- Memory Usage test
- Network Latency
- K8s Node Failure
- Packet Loss
- K8s Pod Unresponsible
